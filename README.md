# Academoscope

B2B SaaS-сервис для аналитики онлайн-курсов.

Этот репозиторий содержит упрощённый, но реальный прототип микросервисной системы:

- FastAPI API Gateway для приёма событий об активности студентов.
- Воркер расчёта метрик на основе сырых событий в PostgreSQL.
- Flask-админка с дашбордом для владельцев курсов.
- Общая библиотека `common` с моделями БД и конфигурацией.

Интеграция с реальными платформами (Teachable, Thinkific и др.), Kafka и Temporal закладывается архитектурно и может быть добавлена на следующем этапе.

## Структура (MVP)

- `common/` — общие модели и подключение к БД.
- `api_gateway/` — FastAPI-сервис для приёма событий.
- `metrics_worker/` — скрипт/сервис для агрегации метрик в отдельные таблицы.
- `dashboard/` — Flask-приложение с HTML-дашбордом.
- `infra/k8s/` — базовые манифесты Kubernetes (Postgres и сервисы).

Подробные инструкции по сборке образов Docker и деплою в Kubernetes будут добавлены после стабилизации каркаса сервисов.
